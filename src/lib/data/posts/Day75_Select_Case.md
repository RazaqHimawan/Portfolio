## Select Case

Mengeksekusi salah satu dari beberapa kelompok pernyataan, bergantung pada nilai ekspresi.

### Syntax

```vb
Select Case testexpression
[ Case expressionlist-n [ statements-n ]]
[ Case Else [ elsestatements ]]
End Select
```
